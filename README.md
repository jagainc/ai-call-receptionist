An intelligent **AI-powered call receptionist** built using **Java Spring Boot** and a modern **React + Vite frontend**, designed to automatically handle customer calls, assist with FAQs, and direct calls to appropriate departments — boosting productivity and reducing manual workload.

---

## ✨ Features

- 📞 **Smart Call Routing** – Uses AI to understand caller intent and direct to the right place.
- 💬 **Conversational UI** – Integrates a natural dialogue system for interaction.
- 🌐 **Frontend:** React + Tailwind CSS for clean, responsive UI.
- 🧠 **Backend:** Spring Boot REST API with NLP processing.
- 🛠️ **Speech Recognition** using Web APIs or integration with external services (Google/DeepSpeech).
- 🗂️ **Modular Design** – Easily extendable components.

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/jagainc/ai-call-assistant.git
cd ai-call-assistant

# Frontend Setup
cd frontend
npm install
npm run dev

# Backend Setup
cd ../backend
./mvnw spring-boot:run
